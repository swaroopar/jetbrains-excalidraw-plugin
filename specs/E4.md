# E4 — Theme sync with the IDE

The editor theme follows the IDE theme (light/dark, via `LafManager` /
`EditorColorsManager`) and updates live when the user switches IDE themes,
matching the VSCode plugin's theme-sync behavior.

## Acceptance Criteria

- AC-E4-01: With a dark IDE theme the editor opens in dark theme; with a light IDE
  theme it opens in light theme.
- AC-E4-02: Switching the IDE theme while the editor is open switches the editor
  theme live without reopening the file.
