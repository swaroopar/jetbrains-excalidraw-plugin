# E8 — Excalidraw file icons

`.excalidraw` and `.excalidraw.png` files display the Excalidraw icon in the IDE
(Project view, editor tabs, navigation, and the File Types settings list) instead
of the generic/unknown-file icon.

The icon is bundled with the plugin as a scalable **SVG** under plugin resources
(e.g. `/icons/excalidraw.svg`, 16×16) and returned from `FileType.getIcon()` for
both `ExcalidrawFileType` and `ExcalidrawPngFileType`, loaded via the IntelliJ
Platform `IconLoader` so it renders correctly on HiDPI displays and adapts to the
IDE theme where applicable.

## Acceptance Criteria

- AC-E8-01: A `.excalidraw` file shows the Excalidraw icon (not the default
  file icon) in the Project view and editor tab.
- AC-E8-02: A `.excalidraw.png` file shows the Excalidraw icon (not the generic
  image icon) in the Project view and editor tab.
- AC-E8-03: The icon resource is bundled in the plugin distribution and both
  `ExcalidrawFileType.getIcon()` and `ExcalidrawPngFileType.getIcon()` return a
  non-null `Icon`.
