# E5 — Export to SVG / PNG

The user can export the current drawing to SVG or PNG using the Excalidraw export
utilities. Export runs locally inside the embedded browser; the generated file is
written to a location chosen via an IDE file chooser.

## Acceptance Criteria

- AC-E5-01: Exporting to SVG produces a valid SVG that visually matches the
  current canvas.
- AC-E5-02: Exporting to PNG produces a valid raster image at the selected scale.
