# C1 — IntelliJ Platform plugin (Gradle + JCEF)

The product is an IntelliJ Platform plugin, implemented in Kotlin and built with
Gradle using the IntelliJ Platform Gradle Plugin
(`org.jetbrains.intellij.platform`). It targets WebStorm and other IntelliJ-
platform IDEs, with a minimum `sinceBuild` at a version where **JCEF** is
available in the JetBrains Runtime.

The Excalidraw web app is bundled as local plugin resources and loaded into JCEF;
no remote backend and no remotely hosted code are introduced. The distributable
is the plugin zip produced by the Gradle `buildPlugin` task, installed via
"Install Plugin from Disk" or the JetBrains Marketplace.
