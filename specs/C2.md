# C2 — CI build & downloadable artifact (GitHub Actions)

A GitHub Actions workflow runs the Gradle `buildPlugin` task and uploads the
resulting plugin distribution zip as a downloadable workflow artifact. It runs on
push and pull request and is also manually triggerable (`workflow_dispatch`), so a
maintainer can download the artifact and install it into the IDE for manual
testing without building locally.

The uploaded artifact is the installable plugin zip (the same build a release
would ship), not raw sources.

## Acceptance Criteria

- AC-C2-01: A GitHub Actions workflow is defined that triggers on `push`,
  `pull_request`, and `workflow_dispatch` and runs `buildPlugin`.
- AC-C2-02: A successful run uploads a downloadable artifact containing the plugin
  distribution zip.
- AC-C2-03: The downloaded zip installs via "Install Plugin from Disk" and opens
  the Excalidraw editor on a `.excalidraw` file.
