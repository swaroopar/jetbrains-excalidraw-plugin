# E2 — Edit the drawing on the embedded canvas

Once a file is open, the user can edit the drawing using the full Excalidraw
canvas running in the JCEF browser (draw shapes, text, freedraw, move, delete,
undo/redo). Scene changes are posted from the web app to Kotlin over the bridge,
which marks the editor document modified so that auto-save (E3) can persist them.

The editor is the official `@excalidraw/excalidraw` component; no editing
capability is reimplemented.

## Acceptance Criteria

- AC-E2-01: Drawing a new shape adds it to the scene and marks the editor document
  modified.
- AC-E2-02: Undo/redo restores the previous/next scene state and updates the
  modified flag accordingly.
